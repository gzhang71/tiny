# Roadmap

Planned and candidate future `tiny_*` projects. Statuses: 🟢 Active · 🟡 WIP · ⚪ Planned / idea.

## Existing

- 🟢 [tiny_ml](https://github.com/gzhang71/tiny_ml) — ML/DL fundamentals from scratch
- 🟢 [tiny_rag](https://github.com/gzhang71/tiny_rag) — retrieval-augmented generation
- 🟢 [tiny_causal_inference](https://github.com/gzhang71/tiny_causal_inference) — causal inference methods
- 🟢 [tiny_rl](https://github.com/gzhang71/tiny_rl) — RL for LLM post-training (SFT, PPO, GRPO, DPO)

## Candidate future projects

- ⚪ **tiny_llm** — pretrain a small language model end to end: tokenizer, data pipeline, training loop, sampling
- ⚪ **tiny_agents** — minimal agent loop: tool use, planning, memory, multi-agent orchestration
- ⚪ **tiny_ab_testing** — experiment design, power analysis, sequential testing, variance reduction (pairs well with tiny_causal_inference)
- ⚪ **tiny_mlops** — model serving, monitoring, feature stores, CI for models, all in miniature
- ⚪ **tiny_recsys** — collaborative filtering, matrix factorization, two-tower models, ranking
- ⚪ **tiny_diffusion** — denoising diffusion models for images, from DDPM up
- ⚪ **tiny_optimization** — convex optimization, gradient methods, and solvers underlying ML
